Chrono更新日志

关闭

版本0.9.0 (2015-09-13)

  • 修复了一个可能引起新任务使用过往任务文件名的bug.
  • 添加了对Data URI scheme和带空格URL的支持。
  • 可以向任务管理页面拖放链接来添加新任务。
  • 支持Ctrl-C拷贝所选中任务的网址。
  • 增强了对媒体资源嗅探的支持。
  • 搜索任务时忽略空格。
  • 在状态栏中显示消息和提示。
    • 添加了按钮以打开或更改默然下载文件夹
    • 将任务清除和排序按钮移至状态栏
  • 诸多优化:
    • 剪贴板的读写速度
    • 处理超长的URL（几十万的长度）
    • 加快任务的载入、排序、过滤、显示更新
    • 加快任务的批量导入、开始、暂停、重启、删除、清理等等
  • 修复了一些其他bug:
    • 重启任务后错误的自动命名
    • 批量描述符中对[9:10]的错误处理
    • 规则系统中重排序后没有进行保存
    • 嗅探器中表格的重排序错误
    • 任务管理页面中删除按钮导致删除错误的行

版本0.8.0 (2015-08-17)

  • 修复了新添下载时出现文件名覆盖的问题。
  • 文件名冲突时添加"跳过"选项。
  • 第一次安装时允许从Chrome导入下载历史。
  • 显示任务错误信息。
  • 修复了规则系统中的一个问题，即datetime.is()函数不起作用。
  • 修复了其他bug.

版本0.7.2 (2015-08-04)

  • 修复了下载危险文件时无法弹出确认对话框的bug.
  • 修复了其他bug.

版本0.7.0 (2015-07-27)

  • 表格模式又回来了。现在在管理器中你可以随时切换表格与列表的管理方式。
  • 改进界面。
  • 解决了新添加任务时的延迟。
  • 解决了若干与任务命名相关的问题。
  • 一些稳定性方面的改进。

版本0.6.0

  • 重新设计下载管理界面，将任务信息面板移至右侧。
  • 支持改变下载管理界面的背景。
  • 添加状态栏，显示全局下载信息。
  • 弹出页面中的面板切换动画更加平滑。
  • 加快弹出页面的弹出速度。
  • 修正了任务进度桌面提醒和任务完成桌面提醒同时出现的问题。
  • 修正了任务完成桌面提醒中图标的不正确显示。
  • 改进了页面上添加一键下载的动画。

版本0.5.3

  • 修正了速度的不正确显示。

版本0.5.2

  • 添加新的命名字段: *text*, *title*, *page_title*.
  • 指定命名方式时支持从最近使用过的命名方式列表中选择。
  • 在资源嗅探器中单击选择或取消所有高亮的链接。
  • 修复了一些小问题。

版本0.5.0

  • 修复了‘无法打开文件’、‘空的新任务对话框’和‘下载.crx/.js/.exe类型文件’等bug。
  • 在弹出页面中显示下载任务列表。
  • 在工具栏图标上显示正在下载任务的数目。
  • 在资源嗅探器中文件大小的滑动条更加平滑。
  • 支持清理任务列表。
  • 优化了弹出页面的弹出速度。

版本0.2.3

  • 改正了规则系统中的一个错误拼写：'r_downloaing'.
  • 修复了用户报告的一些bug。
  • 之前的版本在Linux系统下一直不可用，因为我错误拼写了一些文件路径，现在这个问题解决了。
  • 资源嗅探器中添加了使用文件大小过滤的功能。

版本0.2.2

  • 修复了另存为对话框的一个问题：现在只有点击保存按钮之后下载才会开始。
  • 修复了资源嗅探器的一个bug：无法选中所有行。
  • 修复了一个重新开始任务的bug。
  • 修复了一个任务排序bug。
  • 更好的处理最大“并行下载数目”和“每个服务器连接限制”。

版本0.2.1

  • 修复了一个当插件被停用时出现的bug。
  • 在任务信息面板和桌面提示中预览已完成的图片类型的任务。

版本0.2.0

  • 对Chrono资源嗅探器做了诸多改进:
    在选择下载哪些链接前，你可以预览图片或音频类型的链接，并获知每个链接的文件大小；
    现在你可以下载网页上的媒体文件，如音频、视频流。
  • 显示任务已下载大小。
  • 当任务完成或中断时播放声音提示。
  • 添加简体中文语言支持。
  • 添加两个快捷按钮，让你可以从浏览器的工具栏上直接打开Chrono的管理页面和设置页面，请从下面的链接获取这两个按钮：
    Chrono管理器按钮, Chrono设置按钮.
  • 修复了一些其他问题。

版本0.1.5

  • 新功能：下载剪贴板中的网址，可以在设置»常规»其他一栏中设置。
  • 新功能：在桌面提醒中显示任务进度，可以在设置»界面»桌面提醒一栏中设置。
  • 修复了一些问题；对桌面提醒做了一些改进。
  • 这次更新需要新的权限：访问剪贴板中的内容。

版本0.1.4

  • 移除“一键下载”规则，它现在作为一个选项，可以在设置中开启或关闭。
  • 修复了和新任务对话框相关的一些问题。
